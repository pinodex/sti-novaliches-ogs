array getSheetsContents( array $indices)

Get sheets contents.

Don't be confused with getSheetContents. This method accepts array of sheet indices as a parameter and return an array of sheet contents.

Element keys will be the sheet name and values will be the contents.

This method calls $this->getSheetContents() internally.
